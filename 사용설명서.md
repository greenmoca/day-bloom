# DAY BLOOM 사용설명서 🌸

> 나만의 하루를 피워내는 공간 — 목표·기록·할 일·메모·비전보드·날씨를 한 화면에서.

라이브: https://greenmoca.github.io/day-bloom/

---

## 시작하기

- 브라우저로 위 주소에 접속하면 바로 사용할 수 있어요. 설치 없이 사용 가능합니다.
- 처음 열면 **환영 카드**가 핵심 기능을 짧게 안내해요. (다시 보려면 설정 → 도움말 → "처음 안내 다시 보기")
- 모든 데이터는 **내 브라우저에만** 저장돼요. 서버로 올라가지 않습니다.

## 화면 한눈에 보기

상단에 오늘 날짜와 매일 바뀌는 한 줄 명언이 있고, 아래로 카드들이 이어져요.

- **GOAL** — 올해/이번 달 목표
- **QUICK ADD** — 기록·할 일·메모·목표를 버튼 하나로 추가
- **WEATHER** — 위치 기반 실시간 날씨
- **CALENDAR** — 월간 달력 (할 일 표시 + 감정 색 점)
- **MONTHLY REPORT** — 이번 달 요약 통계
- **DAILY** — 하루 기록 목록
- **MEMO** — 짧은 메모
- **VISION BOARD** — 목표 이미지 모음

## 기능별 사용법

### ⚡ 빠른 추가 (Quick Add)
QUICK ADD 카드의 버튼을 누르면 입력창이 떠요. **오늘 기록 / 할 일 / 메모 / 목표**를 바로 추가할 수 있습니다.

### 🎯 목표 (Goal)
GOAL 카드의 **+** 로 목표를 추가해요. 기간은 올해·이번 달·이번 주 또는 **직접 설정**(시작일·종료일)으로 정할 수 있고, 중요도도 고를 수 있어요. 동그라미를 누르면 완료 체크, 화살표로 순서 변경이 됩니다.

### 📝 하루 기록 (Daily)
제목·감정(😊🙂😪🤔✨)·카테고리·내용·내일 할 일을 적어요. 전체/오늘/이번 달/카테고리별로 걸러 볼 수 있습니다.

### ✅ 할 일 & 📅 캘린더
할 일은 **날짜를 정해** 추가하면 캘린더에 표시돼요. 달력에서 **날짜를 누르면** 그날의 할 일과 기록을 보고 바로 추가할 수 있습니다. 감정은 날짜에 색 점으로 표시돼요.

### 💡 메모
떠오른 생각을 빠르게 적어요. 고정(📌), 상태(임시·보관·완료), 검색을 지원합니다.

### 🖼️ 비전보드 (Vision Board)
카드의 **+** 또는 "이미지 추가"로 사진·GIF를 올려요. 이미지는 자동으로 압축되고(움직이는 GIF는 그대로 유지), 설명 입력·대표 이미지 설정이 가능합니다.

### 🔍 전체 검색
헤더의 **🔍 검색**을 누르면 목표·기록·할 일·메모·비전보드를 한 번에 찾아요. **날짜(예: 2026-06)**, 카테고리, 감정으로도 검색됩니다. 결과를 누르면 해당 항목이 열려요.

### 📊 월간 리포트
이번 달 기록 수, 가장 많이 느낀 감정, 자주 쓴 카테고리, 기록 많은 요일, 완료한 할 일·목표를 요약해 줘요. 일부 지표는 **지난달 대비 증감**도 보여줍니다.

### 🌤️ 날씨
API 키 없이 위치 기반 실시간 날씨가 자동으로 떠요. 매시간 자동 갱신되고, 오프라인이면 마지막으로 본 날씨를 표시합니다. "📍 정확한 위치"를 누르면 더 정확해져요(한 번 허용하면 이후 자동).

## 데이터 관리 (중요)

데이터는 이 브라우저에만 저장되므로 **가끔 백업**하세요.

- **내보내기**: 설정 → 내보내기 → JSON 파일이 저장돼요(이미지 포함).
- **가져오기**: 설정 → 가져오기 → **전체 교체** 또는 **합치기**(중복 없이 새 항목만 추가) 중 선택.
- 7일 넘게 백업을 안 하면 안내가 떠요.
- 기기를 바꾸거나 새 주소로 옮길 땐 **내보내기 → 가져오기**로 데이터를 이사하세요.

## 📲 휴대폰에 앱으로 설치

- **안드로이드(크롬)**: 헤더의 "📲 앱 설치" 버튼을 누르면 홈 화면에 앱으로 설치돼요.
- **아이폰(사파리)**: 공유 메뉴 → "홈 화면에 추가".
- 설치하면 주소창 없는 전체화면 앱처럼 쓸 수 있고, 오프라인에서도 열려요.

## 설정

테마 색(5종), 글씨 크기, 데이터 내보내기/가져오기, 처음 안내 다시 보기를 제공합니다. 테마 색을 바꾸면 탭 아이콘 색도 함께 바뀌어요.

## 자주 묻는 질문

- **다른 기기에서도 보이나요?** 아직은 브라우저별 저장이라, 내보내기/가져오기로 옮겨야 해요.
- **이미지를 많이 올려도 되나요?** 네. 이미지는 별도 저장소(IndexedDB Blob)에 보관돼 수 GB까지 담을 수 있어요.
- **데이터가 사라질까 걱정돼요.** 정기적으로 내보내기로 백업해 두면 안전합니다.
